Favicon Test Page

This page is a test to verify that your favicons are working correctly. You should see your custom favicon in the browser tab.

Favicons in this project:

favicon.ico - The standard favicon that most browsers will use

favicon-32x32.png - 32x32 PNG version

favicon-16x16.png - 16x16 PNG version

apple-touch-icon.png - Used by iOS devices when adding to home screen

Troubleshooting

If you don't see the custom favicon in the browser tab:

  1. Try hard-refreshing the page (Ctrl+F5 or Cmd+Shift+R)
  2. Clear your browser cache
  3. Check if the images above display correctly
  4. Verify that the favicon files are in the correct location
  5. Check the network tab in developer tools to see if the favicon files are being loaded
